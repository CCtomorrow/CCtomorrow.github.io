动态 SQL 通常要做的事情是根据条件包含 where 子句的一部分，mybatis的动态sql还是可以嵌套的。比如：

mybatis大概支持以下几种表达式:

  • if
  • choose (when, otherwise)
  • trim (where, set)
  • foreach

if是一个条件表达语句，但是其实只有if没有else，要表达else可以这样:(当是vip的时候查询女用户，否则查询男用户)

choose是选择类似switch:(当是vip的时候如果是女性查询18岁的用户，男性查询22的用户，否则查询男用户)
